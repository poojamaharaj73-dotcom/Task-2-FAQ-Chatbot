# 🤖 FAQ Chatbot using NLP

## Overview

This project is a simple FAQ Chatbot developed for the **CodeAlpha Artificial Intelligence Internship**. It uses **Natural Language Processing (NLP)** to understand user questions and provides the most relevant answer by matching them with a predefined FAQ dataset.

## Features

* FAQ-based chatbot
* NLP text preprocessing using NLTK
* TF-IDF Vectorization
* Cosine Similarity for question matching
* Simple and interactive Streamlit interface

## Technologies Used

* Python
* Pandas
* NLTK
* Scikit-learn
* Streamlit

## Project Structure

```text
FAQ_Chatbot/
│── app.py
│── chatbot.py
│── faq.csv
│── requirements.txt
│── README.md
```

## How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the application:

```bash
streamlit run app.py
```
